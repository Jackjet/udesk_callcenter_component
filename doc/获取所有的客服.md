获取所有的客服
=========================================

请求地址
-----------------
+ URL '/agent_api/v1/callcenter/agents'
+ VERB GET

请求头
----
```yaml
Authorization: bearer xxxxxxxx
```

返回结果
-----------------

```yaml
code: 编码-返回结果状态
items: List of agents.
    id: 客服ID
    nick_name: 客服姓名
    work_state: 客服在线状态
    work_way: 客服在线方式
    cellphone: 客服手机号
```
